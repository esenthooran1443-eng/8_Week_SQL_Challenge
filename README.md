# 📊 8 Week SQL Challenge

This repository contains my solutions for the **[#8WeekSQLChallenge](https://8weeksqlchallenge.com/)** created by Danny Ma.

The challenge consists of **8 real-world SQL case studies** designed to develop advanced SQL skills and business analysis thinking.

Through this project, I analyze different datasets and solve practical business problems using SQL.

---

## 🧠 Skills Demonstrated

* SQL Data Cleaning
* Data Exploration
* Window Functions
* Common Table Expressions (CTEs)
* Aggregations and Joins
* Business Metrics Analysis
* Customer Behaviour Analysis
* Subscription and Revenue Analysis
